> **모여서 도란도란 코딩👨‍💻👩‍💻** <br/> **개발기간: 2022.06 ~ 2022.11**

### About 모도코

> **모도코**(모여서 도란도란 코딩) 프로젝트는 모각코를 하고 싶은 사람들이 모여 도란도란 코딩할 수 있게 해주는 플랫폼입니다. 개발자 친화적인 기능을 갖추고 모각코로 만들어진 인연이 이어질 수 있게 만들고자 합니다.
